Fundamentos de Python

Passo 1: Introdução ao Python

Python é uma linguagem de programação de alto nível, interpretada e de propósito geral, conhecida por sua sintaxe simples e legibilidade. É amplamente usada para desenvolvimento web, ciência de dados, automação de tarefas e muito mais.

Instalação do Python

No Windows:

Baixe e instale o Python do site oficial: https://www.python.org/

No macOS:

brew install python

No Linux:

sudo apt-get install python3

Passo 2: Primeiros Passos com Python

Executando o Interpretador Python

Abra o terminal ou prompt de comando e digite python ou python3 para iniciar o interpretador Python.

Você verá um prompt interativo onde pode digitar comandos Python e vê-los executados imediatamente.

Escrevendo e Executando Scripts Python

Crie um arquivo com a extensão .py (por exemplo, hello.py) e adicione o seguinte código:

print("Hello, World!")

No terminal, navegue até o diretório onde o arquivo está salvo e execute:

python hello.py

Passo 3: Conceitos Básicos de Programação

Variáveis e Tipos de Dados

Python suporta vários tipos de dados, incluindo números, strings, listas, tuplas e dicionários.


# Exemplo de variáveis
num = 10
nome = "Ana"
lista = [1, 2, 3]
tupla = (4, 5, 6)
dicionario = {"chave": "valor"}

Estruturas de Controle

Utilize estruturas de controle como if, else, elif, for e while para controlar o fluxo do programa.


# Exemplo de estrutura de controle
if num > 5:
    print("Maior que 5")
else:
    print("Menor ou igual a 5")

# Loop for
for i in lista:
    print(i)

# Loop while
count = 0
while count < 5:
    print(count)
    count += 1

Funções e Módulos

Funções permitem reutilizar código. Módulos são arquivos Python que podem conter funções, classes e variáveis.


# Definindo uma função
def saudacao(nome):
    print(f"Olá, {nome}!")

saudacao("Ana")

# Importando um módulo
import math
print(math.sqrt(16))

Passo 4: Trabalhando com Arquivos

Leitura e Escrita de Arquivos

Use as funções open(), read(), write() e close() para manipular arquivos.


# Escrevendo em um arquivo
with open("exemplo.txt", "w") as arquivo:
    arquivo.write("Hello, World!")

# Lendo de um arquivo
with open("exemplo.txt", "r") as arquivo:
    conteudo = arquivo.read()
    print(conteudo)

Manipulação de Arquivos e Diretórios

Utilize o módulo os para manipular arquivos e diretórios.


import os

# Criando um diretório
os.mkdir("novo_diretorio")

# Listando arquivos em um diretório
print(os.listdir("."))

Passo 5: Introdução à Programação Orientada a Objetos (POO)

Classes e Objetos

Classes são "modelos" para criar objetos. Objetos são instâncias de classes.


# Definindo uma classe
class Pessoa:
    def __init__(self, nome, idade):
        self.nome = nome
        self.idade = idade

    def saudacao(self):
        print(f"Olá, meu nome é {self.nome} e eu tenho {self.idade} anos.")

# Criando um objeto
pessoa = Pessoa("Ana", 30)
pessoa.saudacao()

Métodos e Atributos

Métodos são funções definidas dentro de uma classe. Atributos são variáveis associadas a uma classe.

Herança e Polimorfismo

Herança permite que uma classe herde atributos e métodos de outra classe. Polimorfismo permite usar métodos de forma intercambiável em diferentes classes.


# Herança
class Animal:
    def som(self):
        pass

class Cachorro(Animal):
    def som(self):
        return "Latido"

class Gato(Animal):
    def som(self):
        return "Miado"

# Polimorfismo
animais = [Cachorro(), Gato()]
for animal in animais:
    print(animal.som())
