Instalando e Configurando o Termux

Passo 1: Instalar o Termux

1. Abra a Google Play Store no seu dispositivo Android.

2. Pesquise por "Termux" e instale o aplicativo.

3. Abra o Termux após a instalação.

Passo 2: Instalar o Python e Git

1. Atualizar os pacotes do Termux:

pkg update && pkg upgrade

2. Instalar o Python:

pkg install python

Confirme a instalação pressionando y quando solicitado.

3. Instalar o Git:

pkg install git

Passo 3: Criar e Configurar um Ambiente Virtual

1. Instalar o virtualenv:

pip install virtualenv

2. Criar um diretório para o seu projeto:

mkdir meuprojeto

3. Navegar para o diretório do projeto:

cd meuprojeto/

4. Criar um ambiente virtual:

virtualenv env

5. Ativar o ambiente virtual:

. env/bin/activate

Passo 4: Instalar Django e Configurar o Projeto

1. Instalar Django dentro do ambiente virtual:

pip install django

2. Criar um novo projeto Django:

django-admin startproject mysite .

3. Configurar o ambiente criando as tabelas necessárias:

python manage.py migrate

4. Iniciar o servidor de desenvolvimento do Django:

python manage.py runserver

Para parar o servidor, pressione Ctrl + C.

Passo 5: Instalar e Usar o Editor Micro

1. Instalar o Micro:

pkg install micro

2. Criar e editar arquivos com o Micro:

micro nome_do_arquivo.html

Para salvar o arquivo, pressione Ctrl + S.

Para sair do Micro, pressione Ctrl + Q.

Atalhos Comuns no Micro:

  • Salvar: Ctrl + S
  • Sair: Ctrl + Q
  • Copiar: Ctrl + C
  • Cortar: Ctrl + X
  • Colar: Ctrl + V
  • Desfazer: Ctrl + Z
  • Refazer: Ctrl + Y
  • Procurar: Ctrl + F
  • Ir para linha: Ctrl + G

Passo 6: Visualizar Arquivos Estáticos

Se você deseja apenas visualizar arquivos HTML e não configurar todo o projeto Django, pode usar o servidor simples do Python:

python3 -m http.server

Isso inicia um servidor simples que serve arquivos estáticos na pasta atual. Acesse http://localhost:8000 no seu navegador para visualizar o conteúdo.

Passo 7: Configurar Git e GitHub

1. Criar um repositório vazio no GitHub:

  • Acesse GitHub e faça login na sua conta.
  • Clique no botão New (Novo) para criar um novo repositório.
  • Dê um nome para o repositório e clique em Create repository (Criar repositório).

2. Configurar o Git:

git config --global user.name "seu nome"
git config --global user.email "seu email"

3. Gerar e Configurar a Chave SSH:

ssh-keygen -t rsa -b 4096 -C "seu email"

Quando solicitado, pressione Enter para aceitar o local padrão do arquivo.

cat ~/.ssh/id_rsa.pub

Copie o conteúdo exibido.

4. Adicionar a chave SSH ao GitHub:

  • Acesse GitHub e faça login na sua conta.
  • Vá para Settings (Configurações) clicando na sua foto de perfil no canto superior direito.
  • No menu à esquerda, clique em SSH and GPG keys.
  • Clique no botão New SSH key.
  • No campo Title, dê um nome para a chave (por exemplo, "Termux").
  • No campo Key, cole a chave pública que você copiou anteriormente.
  • Clique em Add SSH key para salvar.

Passo 8: Inicializar o Repositório Git e Enviar Arquivos para o GitHub

1. Navegar até o diretório do projeto:

cd ~/meuprojeto

2. Inicializar o repositório Git:

git init

3. Adicionar todos os arquivos ao repositório:

git add .

4. Fazer o primeiro commit:

git commit -m "Primeiro commit"

5. Adicionar o repositório remoto no GitHub:

git remote add origin git@github.com:usuario/nome-do-repositorio.git

6. Enviar os arquivos para o GitHub:

git push -u origin master
